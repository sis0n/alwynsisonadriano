# Project: Alwyn Sison Adriano Portfolio

This project is a personal portfolio website for Alwyn Sison Adriano, a Computer Science student and aspiring Software Developer. It is built using **React**, **TypeScript**, and **Tailwind CSS**.

## Project Structure

- `src/assets/`: Static assets like images and icons.
- `src/components/`: Reusable UI components (Navbar, Hero, Section, Card, etc.).
- `src/data/`: Structured data files (portfolio content).
- `src/docu/`: Source documentation (CV, certificates).
- `src/pages/`: Main page components and routing.
- `src/types/`: TypeScript interfaces and types.
- `public/`: Public assets and `index.html`.

## Tech Stack

- **Framework:** React (Vite/CRA style)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Data Source:** `src/docu/adriano_cv.md`

## Development Guidelines

- **Component-Driven:** Build reusable components for sections.
- **Type Safety:** Ensure all data is typed using interfaces in `src/types/`.
- **Responsive Design:** Use Tailwind CSS for a mobile-first approach.
- **Clean Code:** Follow the conventions observed in the repository.

## Key Commands (Inferred)

- `npm install`: Install dependencies.
- `npm run dev`: Start the development server.
- `npm run build`: Build for production.
